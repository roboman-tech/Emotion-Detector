# Final Project: Emotion Detection

This is the **Final Project** for the emotion detection application using IBM Watson's Natural Language Understanding (NLU) service.

## Project Overview
The application analyzes emotions from text input and classifies the emotions into categories such as joy, sadness, anger, fear, and disgust. It uses the Watson NLU API to perform emotion analysis and can be accessed via a web API built using Flask.

## Features
- Detects emotions from text.
- Provides a web interface for testing the emotion detection.
- Handles errors and invalid inputs gracefully.
- Includes unit tests and static code analysis.

## How to Run
1. Clone the repository.
2. Install dependencies using `pip install -r requirements.txt`.
3. Run the server using `python server.py`.
4. Access the API on `http://127.0.0.1:5000/`.
